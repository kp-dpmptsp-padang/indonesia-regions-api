API gratis untuk data provinsi dan kota/kabupaten di Indonesia

Base URL

https://indonesia-regions-api.vercel.app

Endpoints

1. Mendapatkan Semua Provinsi

GET /provinces

Response:

[
  {
    "id": 1,
    "name": "Aceh",
    "cities": [
      {
        "id": 101,
        "name": "Kota Banda Aceh"
      }
    ]
  }
]

2. Mendapatkan Provinsi Berdasarkan ID

GET /provinces/{id}

Response Success:

{
  "id": 1,
  "name": "Aceh",
  "cities": [
    {
      "id": 101,
      "name": "Kota Banda Aceh"
    }
  ]
}

Contoh Penggunaan

Menggunakan Fetch

// Mengambil semua provinsi
fetch('https://indonesia-regions-api.vercel.app/provinces')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Menggunakan Axios

import axios from 'axios';

// Mengambil provinsi berdasarkan ID
axios.get('https://indonesia-regions-api.vercel.app/provinces/1')
  .then(response => console.log(response.data))
  .catch(error => console.error('Error:', error));

Catatan

  • API ini bersifat gratis dan dapat digunakan untuk keperluan development
  • Data mencakup 38 provinsi di Indonesia termasuk pemekaran terbaru
  • Gunakan API dengan bijak dan terapkan caching untuk mengurangi beban server
